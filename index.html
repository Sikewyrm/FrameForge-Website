Everything you need to forge your arsenal

FrameForge runs alongside your game and keeps you informed — no manual entry, no spreadsheets.

📦

Live Inventory

Resources, blueprints, weapons, Warframes — all tracked automatically as your inventory changes in game.

🔨

Foundry Tracker

See which components you own, which blueprints you have, and exactly what you're still missing for every build.

💎

Relic Indicators

Know which relics drop the parts you need, right in the component list — no more switching tabs.

⭐

Mastery Rank

Your current Mastery Rank is always visible in the header — updated every scan cycle.

🔒

Vault Status

Each prime item shows whether it's currently vaulted or available in the active drop table.

⚡

Read-Only

FrameForge never writes to or modifies your Warframe installation — it only reads.

Simple by design

Get started in seconds.

1

Download and install

Download the latest release and run the installer. No configuration needed.

2

Launch Warframe

Start Warframe as you normally would. FrameForge detects the game process automatically.

3

Return to your Orbiter

After a mission or visiting a relay, come back to your Orbiter — your inventory refreshes automatically.
